## Plano de Estudo – 4 semanas  
**Tecnologias**: React, Angular e JavaScript  
**Tempo diário**: 2 h (4 dias/semana) → 8 h/semana  
**Foco**: prática, projetos e misto (teoria + hands‑on)  

| Semana | Tema Principal | Objetivos de Aprendizagem | Atividades (2 h/dia) | Projeto/Prática | Recursos Sugeridos |
|--------|----------------|---------------------------|----------------------|-----------------|--------------------|
| **1** | JavaScript ES6+ <br> (variáveis, funções, módulos, async/await) | Dominar a sintaxe moderna e padrões de código | 1️⃣ Revisar conceitos básicos  <br> 2️⃣ Exercícios de “code‑wars”  <br> 3️⃣ Refatorar um script antigo | **Mini‑app**: Calculadora simples (JS puro) | MDN, *Eloquent JavaScript*, Exercism |
| **2** | React – Fundamentos | Entender JSX, componentes, props, state, ciclo de vida | 1️⃣ Construir componentes funcionais  <br> 2️⃣ Gerenciar estado local  <br> 3️⃣ Consumir API REST (fetch) | **Projeto**: Lista de tarefas (To‑Do) com CRUD | React Docs, *React – The Complete Guide* (Udemy) |
| **3** | Angular – Fundamentos | Estrutura de módulos, componentes, serviços, DI, rotas | 1️⃣ Criar um módulo e componentes  <br> 2️⃣ Implementar rotas  <br> 3️⃣ Consumir API com HttpClient | **Projeto**: Blog simples (posts CRUD) | Angular Docs, *Angular – The Complete Guide* (Udemy) |
| **4** | Integração + Projeto Final | Combinar React + Angular + JS puro em um workflow | 1️⃣ Refatorar a To‑Do em React  <br> 2️⃣ Refatorar o Blog em Angular  <br> 3️⃣ Criar um **dashboard** que consome ambos os back‑ends | **Projeto Final**: Dashboard de produtividade (React front + Angular API) | GitHub, Netlify, Vercel |

### Dicas de Organização

| Dica | Por que funciona |
|------|------------------|
| **Divida o tempo**: 30 min de revisão + 90 min de prática | Mantém a atenção e reforça o aprendizado |
| **Use controle de versão**: Git | Facilita rollback e colaboração |
| **Documente**: README, comentários | Ajuda a lembrar e compartilhar |
| **Teste**: Jest (React) / Karma (Angular) | Garante qualidade e confiança |
| **Revisite**: Ao fim de cada semana, faça um resumo rápido | Consolida o que foi aprendido |
